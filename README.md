## 🚀 Featured Projects

### 🔬 Drug Consumption & Personality Profiling
ML classification pipeline on the UCI Drug Consumption dataset (1,877 respondents), predicting Cannabis, Cocaine, and Heroin usage. Engineered 5 composite features, handled 85:15 class imbalance with SMOTE, and improved F1-macro by up to **17.7%** using Logistic Regression & Random Forest under 5-fold Stratified CV.  
`Python` `scikit-learn` `imbalanced-learn` `pandas` `statsmodels`

### 🔐 Secure Vault — Encrypted File Sharing
Zero-trust full-stack web app with AES file encryption, JWT + bcrypt auth, role-based access control, and time-expiring share links. RESTful Express.js backend with Multer and SQLite.  
`React` `Node.js` `Express.js` `SQLite` `AES Encryption`
